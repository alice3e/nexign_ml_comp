# Процесс глубоководного исследования океана

Исследовательское судно прибывает в целевую точку, команда развертывает автономный подводный аппарат (AUV) с предварительной проверкой систем жизнеобеспечения, навигации, связи и научного оборудования. Параллельно гидролокатор сканирует дно для построения батиметрической карты и выбора точек интереса. AUV погружается по заданной траектории с постоянной корректировкой по инерциальной навигации и акустическим маякам. На глубине аппарат проводит фото/видеосъемку, отбор проб грунта и воды, измерение температуры, солености и химического состава. При обнаружении гидротермальных источников или биологических объектов запускается детальное обследование с использованием манипулятора. Система мониторит давление корпуса, заряд батарей и расход балласта, при критических значениях инициирует всплытие. Данные передаются на судно через акустический модем с низкой пропускной способностью, полный объем загружается после всплытия. При потере связи AUV автоматически всплывает в заданную точку. Используй 6 лейнов (Судно, AUV, Навигация, Научное оборудование, Связь, Система безопасности), подпроцесс для погружения/всплытия, Parallel Gateway для параллельных измерений, XOR Gateway для режимов исследования, Error Boundary Event для аварийного всплытия, Event-Based Gateway для обнаружения объектов интереса.
