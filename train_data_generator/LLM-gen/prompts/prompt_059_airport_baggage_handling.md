# Процесс обработки багажа в аэропорту

Пассажир регистрирует багаж на стойке, система печатает бирку с штрих-кодом и отправляет чемодан на конвейер. Багаж проходит через сканер безопасности, при обнаружении подозрительных предметов направляется на дополнительный досмотр. Параллельно система определяет рейс и терминал назначения, сортировочная система автоматически направляет багаж по конвейерам к нужному выходу на посадку. Грузчики загружают багаж в контейнеры и на борт самолета. При пересадке багаж автоматически перенаправляется на стыковочный рейс. После прилета багаж выгружается и доставляется на ленту выдачи. Если багаж потерян - запускается процесс розыска с уведомлением пассажира. Используй 6 лейнов (Регистрация, Безопасность, Сортировка, Погрузка, Система трекинга, Розыск), Parallel Gateway для параллельной сортировки и проверки, XOR Gateway для обработки подозрительного багажа, подпроцесс для пересадки, Error Event для потери багажа.
