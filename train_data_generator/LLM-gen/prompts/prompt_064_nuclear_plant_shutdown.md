# Процесс аварийной остановки атомной электростанции

Система мониторинга детектирует критическое отклонение параметров реактора (температура, давление, радиация), автоматически инициируется SCRAM - аварийная остановка. Параллельно запускаются системы аварийного охлаждения активной зоны, изолируются контуры циркуляции, активируются резервные дизель-генераторы. Операторы в главном щите управления подтверждают срабатывание защит и контролируют параметры, при необходимости переходят на ручное управление. Одновременно оповещаются регулирующие органы и аварийные службы, эвакуируется персонал из опасных зон. Система непрерывно мониторит уровень радиации и состояние защитных барьеров, при их нарушении эскалирует до полной эвакуации. Используй 6 лейнов (Автоматика, Операторы, Охлаждение, Энергоснабжение, Безопасность, Регулятор), Event-Based Gateway для детекции аварии, Parallel Gateway для одновременных защитных действий, подпроцесс для процедуры SCRAM, Escalation Event для повышения уровня аварии, Error Boundary Event для отказа систем безопасности.
