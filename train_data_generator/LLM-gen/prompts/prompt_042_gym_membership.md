# Процесс оформления абонемента в фитнес-клуб

Создай BPMN диаграмму процесса покупки абонемента в спортзал. Клиент выбирает тип абонемента, менеджер проводит экскурсию по клубу, клиент заполняет анкету здоровья, оплачивает абонемент, получает карту доступа и проходит вводный инструктаж. Используй 2 пула (Клиент, Фитнес-клуб с лейнами), XOR Gateway для выбора типа абонемента, Parallel Gateway для параллельного оформления карты и инструктажа.
