# Процесс производства лекарственного препарата по GMP

Производство начинается с получения одобренной серии от отдела контроля качества и резервирования сырья. Параллельно проводится валидация оборудования, подготовка чистых помещений и проверка квалификации персонала. Операторы выполняют производственные операции строго по утвержденной технологической инструкции с документированием каждого шага в batch record. На критических этапах проводится in-process контроль качества, при отклонении параметров серия останавливается для расследования. После завершения производства образцы отправляются в лабораторию для полного анализа качества (идентификация, чистота, растворение, стабильность). Параллельно проверяется документация на полноту и соответствие. При положительном заключении QC серия выпускается Уполномоченным лицом, при отклонении - утилизируется с расследованием причин. Используй 6 лейнов (Планирование, Производство, QC лаборатория, QA, Склад, Уполномоченное лицо), подпроцесс для производственного цикла, Parallel Gateway для параллельных проверок, XOR Gateway для решения по качеству, Error Event для отклонений, подпроцесс для расследования deviation.
