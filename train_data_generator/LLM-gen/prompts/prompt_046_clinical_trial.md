# Процесс проведения клинического исследования лекарства

Пациент проходит скрининг на соответствие критериям, подписывает информированное согласие, получает препарат или плацебо (рандомизация), проходит регулярные обследования с фиксацией побочных эффектов. При серьезных нежелательных явлениях пациент выводится из исследования с экстренным уведомлением этического комитета. Используй 4 пула (Пациент, Исследователь, Лаборатория, Этический комитет), Event-Based Gateway для рандомизации, подпроцесс с циклом для регулярных визитов, Error Boundary Event для обработки серьезных побочных эффектов, Message Event для коммуникации с комитетом.
