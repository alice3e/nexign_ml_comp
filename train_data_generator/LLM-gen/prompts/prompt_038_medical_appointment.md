# Процесс записи на прием к врачу

Создай BPMN диаграмму процесса записи к врачу в поликлинике. Пациент записывается через регистратуру или онлайн, система проверяет доступность врача, пациент приходит на прием, врач проводит осмотр и назначает лечение. Используй 3 пула (Пациент, Регистратура, Врач), XOR Gateway для выбора способа записи, Message Event для напоминаний.
