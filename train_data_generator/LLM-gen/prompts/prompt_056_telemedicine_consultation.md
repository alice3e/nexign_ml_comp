# Процесс телемедицинской консультации

Пациент записывается на онлайн-консультацию через платформу, заполняет анамнез и загружает результаты анализов. Система проверяет доступность врача и отправляет напоминание обеим сторонам. В назначенное время проводится видеоконсультация, врач изучает документы, проводит опрос и при необходимости назначает дополнительные обследования или выписывает электронный рецепт. Параллельно ведется запись консультации и автоматическая транскрипция. После консультации формируется медицинское заключение, пациент оплачивает услугу и может оставить отзыв. Используй 3 пула (Пациент, Платформа, Врач), подпроцесс для видеоконсультации, Parallel Gateway для записи и транскрипции, XOR Gateway для назначения дополнительных процедур, Message Event для напоминаний и уведомлений.
