# Процесс стыковки космического корабля с МКС

Корабль выходит на орбиту сближения с МКС, бортовой компьютер рассчитывает серию маневров для постепенного сближения с контролем расхода топлива. Параллельно экипаж МКС готовит стыковочный узел, проверяет герметичность и активирует систему захвата. На расстоянии 200м включается автоматическая система сближения KURS, использующая радиолокацию для точного наведения. Корабль выполняет серию коррекций траектории, постоянно мониторя относительную скорость и угол подхода. На финальном этапе скорость сближения снижается до 0.1 м/с, система выравнивает оси стыковочных узлов с точностью до градуса. При контакте срабатывают защелки, создается герметичное соединение, проводится проверка на утечки воздуха. Параллельно соединяются электрические и гидравлические интерфейсы для передачи энергии и ресурсов. При обнаружении проблем на любом этапе инициируется процедура отхода на безопасное расстояние для повторной попытки. После успешной стыковки выравнивается давление, открываются люки и экипажи встречаются. Используй 7 лейнов (Корабль, МКС, Навигация, KURS, Стыковочный механизм, Контроль герметичности, ЦУП), подпроцесс для серии маневров сближения, Parallel Gateway для параллельных проверок, XOR Gateway для go/no-go решений, Error Boundary Event для abort процедуры, подпроцесс с циклом для коррекций траектории.
