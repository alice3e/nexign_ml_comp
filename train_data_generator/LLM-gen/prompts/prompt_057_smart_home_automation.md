# Процесс автоматизации умного дома при уходе жильца

Система детектирует уход последнего жильца через датчики движения и геолокацию смартфонов. Запускается сценарий "Никого нет дома": параллельно выключается освещение во всех комнатах, регулируется температура до экономного режима, активируется охранная сигнализация и камеры видеонаблюдения. Если обнаружены открытые окна - отправляется уведомление владельцу. При срабатывании датчиков дыма или протечки воды система экстренно отключает соответствующие системы и вызывает аварийные службы. При возвращении жильца сценарий отменяется и восстанавливаются комфортные настройки. Используй 5 лейнов (Датчики, Контроллер, Освещение, Климат, Безопасность), Event-Based Gateway для детекции ухода, Parallel Gateway для параллельного управления системами, Error Boundary Event для аварийных ситуаций, Compensation Event для отмены сценария.
