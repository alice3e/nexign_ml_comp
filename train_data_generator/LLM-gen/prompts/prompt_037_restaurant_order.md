# Процесс заказа еды в ресторане

Создай BPMN диаграмму процесса заказа в ресторане. Гость делает заказ официанту, кухня готовит блюда, официант подает заказ, гость оплачивает счет. Используй 3 лейна (Гость, Официант, Кухня), Parallel Gateway для параллельного приготовления блюд, Timer Event для контроля времени приготовления.
