# Процесс одобрения ипотечного кредита

Клиент подает заявку с выбором объекта недвижимости, банк проверяет кредитную историю и платежеспособность, параллельно оценщик проводит оценку недвижимости, юристы проверяют юридическую чистоту сделки. При положительном решении всех проверок готовится кредитный договор, проводится страхование, регистрация залога в Росреестре и выдача кредита. Используй 5 лейнов (Клиент, Кредитный отдел, Оценщик, Юридический отдел, Росреестр), Parallel Gateway для параллельных проверок, подпроцесс для оформления документов, XOR Gateway для принятия решения, Escalation Event при отклонении заявки.
