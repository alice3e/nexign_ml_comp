# Процесс трансплантации органа

Донорский орган становится доступен, координационный центр запускает поиск совместимого реципиента по базе данных с учетом группы крови, HLA-типирования, срочности и времени ожидания. Параллельно проводится медицинское обследование донора и типирование органа, организуется логистика транспортировки с контролем времени ишемии. Найденному реципиенту экстренно сообщают о доступности органа, он прибывает в клинику для предоперационной подготовки. Хирургические бригады параллельно проводят изъятие органа у донора и подготовку реципиента к операции. Орган транспортируется в специальном контейнере с мониторингом температуры, проводится трансплантация с постоянным контролем жизненных показателей. После операции начинается иммуносупрессивная терапия и мониторинг отторжения. Используй 7 лейнов (Координация, Донор, Реципиент, Лаборатория, Логистика, Хирургия, Реанимация), Event-Based Gateway для поиска реципиента, Parallel Gateway для параллельных операций, Timer Boundary Event для контроля времени ишемии, подпроцесс для трансплантации, Error Event для осложнений.
