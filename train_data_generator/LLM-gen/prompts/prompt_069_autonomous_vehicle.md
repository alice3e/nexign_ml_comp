# Процесс автономного вождения автомобиля

Система восприятия непрерывно обрабатывает данные с камер, лидаров, радаров и GPS, создавая 3D карту окружения с детекцией объектов (автомобили, пешеходы, знаки, разметка). Модуль локализации определяет точное положение на HD-карте с точностью до сантиметров. Параллельно система планирования строит траекторию движения с учетом правил дорожного движения, предсказания поведения других участников и оптимизации комфорта. Контроллер управления отправляет команды на рулевое управление, тормоза и акселератор с частотой 100 Гц. При обнаружении критической ситуации (внезапное препятствие, отказ датчика, потеря связи) активируется система безопасной остановки с передачей управления водителю или автоматической парковкой. Все действия логируются в черный ящик для анализа инцидентов. Используй 7 лейнов (Восприятие, Локализация, Планирование, Предсказание, Управление, Безопасность, Телеметрия), Parallel Gateway для параллельной обработки сенсоров, подпроцесс с циклом для планирования траектории, Event-Based Gateway для детекции опасности, Error Boundary Event для отказов систем, Escalation Event для передачи управления водителю.
