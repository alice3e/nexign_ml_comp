# Процесс генной терапии пациента

Пациент проходит генетическое тестирование для подтверждения мутации и определения подходящего вектора доставки (AAV, лентивирус, CRISPR). Параллельно производится персонализированный терапевтический вектор с целевым геном, проводится контроль качества на стерильность, титр и чистоту. Пациент проходит предварительную иммуносупрессию для снижения иммунного ответа на вектор. В день терапии проводится премедикация, вектор вводится внутривенно или локально в целевой орган под контролем витальных функций. Параллельно мониторятся показатели крови, иммунный ответ и признаки цитокинового шторма. В первые недели пациент находится под наблюдением с регулярными анализами на экспрессию трансгена и побочные эффекты. Через месяц проводится биопсия для оценки эффективности встраивания гена. При недостаточной эффективности рассматривается повторное введение или альтернативная терапия. Используй 7 лейнов (Генетик, Производство, Лаборатория, Клиника, Мониторинг, Иммунология, Патоморфология), Parallel Gateway для параллельного производства и подготовки пациента, подпроцесс для процедуры введения, XOR Gateway для оценки эффективности, Error Boundary Event для тяжелых побочных эффектов, подпроцесс с циклом для долгосрочного наблюдения.
