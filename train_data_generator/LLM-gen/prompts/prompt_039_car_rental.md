# Процесс аренды автомобиля

Создай BPMN диаграмму процесса аренды автомобиля. Клиент выбирает автомобиль и даты, система проверяет доступность, клиент оформляет договор и вносит залог, получает автомобиль, возвращает его и получает залог обратно. Используй 2 пула (Клиент, Компания), XOR Gateway для проверки доступности и состояния автомобиля при возврате, Parallel Gateway для оформления документов и проверки автомобиля.
