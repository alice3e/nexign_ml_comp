# Процесс записи на онлайн-курс

Создай BPMN диаграмму процесса регистрации на онлайн-курс. Студент выбирает курс, регистрируется, оплачивает обучение, получает доступ к материалам, проходит уроки и тесты, получает сертификат при успешном завершении. Используй 2 пула (Студент, Платформа), XOR Gateway для проверки оплаты и результатов тестов, Timer Event для дедлайнов заданий, Message Event для напоминаний.
