# Процесс подачи и экспертизы патентной заявки

Заявитель подает заявку с описанием изобретения и формулой, патентное ведомство проводит формальную экспертизу на соответствие требованиям. Параллельно проводится поиск аналогов в базах данных и проверка новизны. Эксперт проводит содержательную экспертизу, при выявлении замечаний отправляет запрос заявителю на уточнение. После устранения замечаний и положительного заключения публикуется решение о выдаче патента. При отрицательном решении заявитель может подать апелляцию. Используй 3 пула (Заявитель, Патентное ведомство, Апелляционная комиссия), подпроцесс с циклом для итераций доработки, Parallel Gateway для параллельных проверок, XOR Gateway для решения по результатам экспертизы, Escalation Event для апелляции.
