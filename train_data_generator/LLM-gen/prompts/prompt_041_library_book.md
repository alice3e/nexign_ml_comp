# Процесс выдачи книги в библиотеке

Создай BPMN диаграмму процесса выдачи книги в библиотеке. Читатель ищет книгу в каталоге, библиотекарь проверяет наличие и задолженности читателя, выдает книгу на срок, читатель возвращает книгу, система проверяет срок возврата и начисляет штраф при просрочке. Используй 2 пула (Читатель, Библиотека), XOR Gateway для проверки наличия книги и просрочки, Timer Event для контроля срока возврата.
