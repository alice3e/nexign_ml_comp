# Процесс заказа такси через приложение

Создай BPMN диаграмму процесса вызова такси. Пассажир указывает адреса, система ищет ближайшего водителя, водитель принимает заказ, едет к пассажиру, везет по маршруту, пассажир оплачивает поездку и оценивает водителя. Используй 3 пула (Пассажир, Система, Водитель), XOR Gateway для поиска водителя, Message Event для уведомлений о статусе, Timer Event для отмены заказа при долгом ожидании.
