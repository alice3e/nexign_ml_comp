# Процесс бронирования авиабилета

Создай BPMN диаграмму процесса покупки авиабилета. Пассажир ищет рейс, выбирает место, вводит данные, оплачивает билет, получает электронный билет и проходит регистрацию на рейс. Используй 3 пула (Пассажир, Система бронирования, Авиакомпания), XOR Gateway для проверки доступности мест, Timer Event для автоматической отмены неоплаченной брони через 30 минут.
