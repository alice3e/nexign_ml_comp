# Процесс производства продукции на заводе

Производственный заказ проходит через планирование, закупку материалов, производство, контроль качества и отгрузку. Параллельно идет проверка наличия сырья на складе и резервирование производственных мощностей. При обнаружении брака продукция возвращается на доработку или утилизируется. Используй 4 лейна (Планирование, Закупки, Производство, Контроль качества), подпроцесс для производственного цикла, XOR Gateway для решения по браку, Parallel Gateway для параллельных проверок, Timer Boundary Event для контроля сроков производства.
