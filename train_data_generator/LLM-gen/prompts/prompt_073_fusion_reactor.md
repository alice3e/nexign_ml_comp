# Процесс запуска термоядерного реактора

Система подготовки плазмы откачивает вакуумную камеру до сверхнизкого давления, параллельно сверхпроводящие магниты охлаждаются жидким гелием до 4K и заряжаются до рабочего тока для создания тороидального магнитного поля. Инжекторы нейтральных пучков и гиротроны для нагрева плазмы проходят предстартовую диагностику. Система подачи топлива готовит смесь дейтерия и трития в точных пропорциях. После достижения готовности всех систем запускается пробой газа для создания начальной плазмы, затем включаются системы дополнительного нагрева до температуры зажигания термоядерной реакции (150 млн градусов). Параллельно множество диагностических систем мониторят параметры плазмы (температура, плотность, примеси, MHD-нестабильности), система управления в реальном времени корректирует форму плазмы и подачу топлива. При детекции срыва плазмы (disruption) система экстренно вводит примеси для мягкой остановки и защиты стенок камеры. После достижения стационарного режима начинается выработка энергии с отводом тепла через бланкет и генерацию электричества. Используй 8 лейнов (Вакуум, Магниты, Нагрев, Топливо, Диагностика, Управление, Защита, Энергосистема), Parallel Gateway для параллельной подготовки систем, подпроцесс для поддержания плазмы, Error Boundary Event для disruption, XOR Gateway для режимов работы, подпроцесс с циклом для стационарного горения.
