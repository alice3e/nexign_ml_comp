# Процесс бронирования номера в отеле

Создай BPMN диаграмму процесса бронирования номера в отеле. Гость выбирает даты и номер, система проверяет доступность, гость оплачивает (онлайн или при заезде), получает подтверждение. Используй 3 пула (Гость, Система, Отель), XOR Gateway для проверки доступности и способа оплаты, Message Event для уведомлений.
