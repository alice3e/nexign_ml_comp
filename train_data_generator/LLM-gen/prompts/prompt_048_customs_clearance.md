# Процесс таможенного оформления импортного груза

Декларант подает таможенную декларацию с документами, таможня проверяет документы и классифицирует товар, параллельно проводится досмотр груза и проверка соответствия санитарным нормам. При обнаружении нарушений груз задерживается для дополнительной проверки или возвращается отправителю. После оплаты пошлин и сборов выдается разрешение на выпуск товара. Используй 4 лейна (Декларант, Таможенный инспектор, Досмотр, Лаборатория), Parallel Gateway для параллельных проверок, XOR Gateway для решения по результатам проверки, подпроцесс для процедуры досмотра, Conditional Event для выборочного досмотра.
