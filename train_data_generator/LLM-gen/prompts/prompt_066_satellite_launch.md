# Процесс запуска спутника на орбиту

Центр управления полетом проводит финальную проверку готовности спутника, ракеты-носителя и наземных систем за T-24 часа. Параллельно метеослужба мониторит погодные условия, служба безопасности очищает зону запуска. При обнаружении критических проблем запуск откладывается на резервное окно. В T-10 минут начинается автоматическая предстартовая последовательность: заправка топливом, герметизация, перевод на автономное питание. Система автоматически проверяет тысячи параметров, при отклонениях запускается процедура abort с безопасным сливом топлива. После старта ракета проходит этапы отделения ступеней, раскрытия обтекателя, выхода на орбиту и отделения спутника. Спутник разворачивает солнечные панели, устанавливает связь с Землей и начинает орбитальные маневры для выхода на целевую орбиту. Используй 6 лейнов (ЦУП, Стартовый комплекс, Ракета, Спутник, Телеметрия, Служба безопасности), подпроцесс для предстартовой последовательности, Parallel Gateway для параллельных проверок, XOR Gateway для go/no-go решений, Error Boundary Event для abort процедуры, Timer Event для окон запуска.
