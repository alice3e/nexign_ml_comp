# Процесс стриминга видео с адаптивным качеством

Пользователь запускает видео, CDN определяет ближайший сервер и начинает потоковую передачу. Система мониторит скорость соединения и буферизацию, динамически переключая качество между 4K/1080p/720p/480p. Параллельно загружаются субтитры на выбранном языке и собирается аналитика просмотра. При обрыве соединения автоматически возобновляется воспроизведение с последней позиции. Рекламные вставки загружаются заранее и показываются в определенные моменты с возможностью пропуска через N секунд. Используй 4 лейна (Клиент, CDN, Аналитика, Рекламная система), Event-Based Gateway для адаптации качества, Parallel Gateway для параллельной загрузки контента, Error Boundary Event для обработки обрывов, Timer Event для рекламных пауз, подпроцесс для буферизации.
